# Community Surplus

Community Surplus is a full-stack web application that provides a community-driven marketplace for buying and selling surplus or pre-owned products. The platform promotes sustainable consumption by encouraging users to reuse items that are still in good condition, helping reduce waste while making quality products more affordable and accessible.

## ✨ Features

* Secure user registration and login
* Create, update, and manage product listings
* Upload product images with pricing and location details
* Browse and search available products
* Save products to a personal wishlist
* Contact sellers directly
* Responsive and user-friendly interface

## 🛠️ Tech Stack

### Frontend

* HTML5
* CSS3
* JavaScript

### Backend

* Node.js
* Express.js

### Database

* MySQL

### Deployment

* Vercel (Frontend)
* Render (Backend)

### Version Control

* Git & GitHub

## 📌 Conclusion

Community Surplus is a full-stack marketplace application that demonstrates practical web development skills, including user authentication, CRUD operations, database integration, and responsive UI design. By enabling users to buy and sell surplus products within their community, the project promotes sustainable consumption while showcasing the integration of modern frontend, backend, and database technologies.
